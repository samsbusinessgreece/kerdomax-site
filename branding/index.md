# KERDOMAX Branding Kit

This folder contains the branding assets and guidelines to implement the KERDOMAX brand across your site and materials.

Files included:
- branding/logo-axis.svg
- branding/logo-text.svg
- branding/favicon.svg
- branding/colors.md
- branding/usage.md

How to use:
- Use the axis logo with the wordmark in a left-to-right lockup for headers and hero. The palette below provides core colors.
- Keep a premium SaaS aesthetic: deep navy backgrounds, slate blue typography, warm beige accents, and teal action highlights.
- Use Montserrat for headings; system font for body text to ensure fast load times.

"KerdoMax" brand is defined by a precise 8x8 grid, strong negative space, and consistent color rules as described in the dedicated files.
